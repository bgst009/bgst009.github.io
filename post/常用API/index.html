常用API

API

概述

API(Application Programming Interface)，应用程序编程接口。Java API是一本程序员的 字典 ，是JDK中提供给 我们使用的类的说明文档。这些类将底层的代码实现封装了起来，我们不需要关心这些类是如何实现的，只需要学 习这些类如何使用即可。所以我们可以通过查询API的方式，来学习Java提供的类，并得知如何使用它们。

API使用过程

  1. 打开帮助文档。
  2. 点击显示，找到索引，看到输入框。
  3. 你要找谁？在输入框里输入，然后回车。
  4. 看包。java.lang下的类不需要导包，其他需要。
  5. 看类的解释和说明。
  6. 学习构造方法。
  7. 使用成员方法

static静态

关键字概述

  • 一个类成员变量或者方法，一旦使用static 那么其内容不在属于 对象 ，而是属于 类本身
  • 凡是本类对象，共享同一份 数据

package cn.itcast.day07.static_demo;

public class Student {
    private  String name;
    private int age;
    private int id;
    static String room;
    private static int id_counter=0;

    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }

    public Student(String name, int age) {
        this.name = name;
        this.age = age;
        this.id=++id_counter;
    }

    public Student() {
        id_counter++;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
}

包装类

把基本数据类型打包就可以使用相应的方法

  • int ===> Integer

  • char ==> Character

  • 其余全部为首字母大写

装箱

//装箱
        //构造方法
        Integer in1 = new Integer(5);
        Integer in2 = new Integer("4");

        System.out.println("in1: "+in1+" in2: "+in2);

        //静态方法
        Integer in3 = Integer.valueOf(1);
        System.out.println("in3: "+in3);

拆箱

 //拆箱
        int int1=in1.intValue();
        int int2=in2.intValue();
        int int3=in3.intValue();

        System.out.println("in1: "+in1+" in2: "+in2);
        System.out.println("in3: "+in3);

自动拆箱于装箱

public static void main(String[] args) {
        Integer integer = new Integer(5);

        ArrayList<Integer> al1=new ArrayList<>(1);

基本数据类型于字符串之间的相互转换

    public static void main(String[] args) {
        //基本类型转换成字符串
            //1.基本类型的值+“string”
        int int1 = 100;
        String s1 = int1 + "200";
        System.out.println("s1: "+s1);
            //包装类的静态方法toString(args)
        String s2 = Integer.toString(100);
        System.out.println("s2: "+s2+100);
            //string类的静态方法valueOf()
        String s3=String.valueOf(200);
        System.out.println("s3: "+s3+100);

        //String-->基本类型
            //使用包装类的静态方法paresXXX()
        int i = Integer.parseInt(s1);
        System.out.println("s1: "+(i-200));

    }
