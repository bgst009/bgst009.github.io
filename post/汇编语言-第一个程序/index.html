2020-02-17 1 min read # 归纳

第一个程序

编译和链接

  1. 编译 masm .mas --> .obj
  2. 链接 link .obj --> .exe

exe 文件 的描述信息中 保存的程序入口 地址
然后 系统 通过 描述文件 来设置 cs:ip 和 其它内存

asm 文件 -- 汇编语言（1.汇编指令2.伪指令3.符号体系）

  1. 汇编指令 由编译器 翻译成010101 的机器指令 最后由 CPU 执行
  2. 伪指令和符号体系 由编译器执行
  • 程序返回功能
    把系统加载程序的时候 给程序分配的内存 ， 设置的寄存器 返还给系统，因为 系统资源 是有限的
 mov ax,4c00
 int 21H

程序的跟踪 debug + 程序名

  • p 执行 int 指令

  • q 退出

  • cx == 程序长度

  • PSP区 从 ds:0 开始的 256 个字节

快速编译

  • 字母型数字前面 必须加 0；

默认代码（目前）

assum cs:code

code segment

     ;填写内容

     mov ax,4c00H
     int 21H

code ends

end
