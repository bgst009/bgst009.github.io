第13章 int 指令

属于内中断的一种

int指令

  • 指令格式

    int n

  • 执行过程

    1. 获取中断类型码 n
    2. 标志寄存器入栈，IF = 0，TF = 0
    3. CS:IP 入栈
    4. CS:IP 指向中断向量表中的中断地址
    5. 转去执行终端程序

供程序调用的中断例程

  • int 和 iret 配合使用 类似 call 和 ret
  • 注意把中断程序中用到的寄存器压到栈中，避免寄存器冲突
